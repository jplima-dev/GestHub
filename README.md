# **GestHub**

This is a school project developed by my brother and me. **GestHub** is a web application designed to make your daily life as a manager a whole lot easier, more organized, and efficient.

##  About the Project
Our goal with this project was to apply what we've learned in school to build something practical and useful. **GestHub** streamlines tasks and helps you manage your daily activities without the headache.
