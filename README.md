# **GestHub**

This is a school project developed by my brother and me. **GestHub** is a web application designed to make your daily life as a manager a whole lot easier, more organized, and efficient.

##  About the Project
Our goal with this project was to apply what we've learned in school to build something practical and useful. **GestHub** streamlines tasks and helps you manage your daily activities without the headache.

##  How To Use

Open the terminal in your project directory and execute these commands:

1. **Activate the venv**:
   ```bash
   .venv\Scripts\activate.bat **windows
   .venv\bin\activate         **linux
   
2. **Install the dependencies** (if you have a `requirements.txt` file):
   ```bash
   pip install -r requirements.txt

3. **Initialize the db**:
   ```bash
   python scripts\init_db.py

4. **Run the project and open the website**:
   ```bash
   uvicorn backend.app.main:app --reload
   INFO: Uvicorn running on http://127.0.0.1:8000
