O que este cadastro faz

  • 1
    Cria o acesso do usuário para entrar no sistema.
  • 2
    Associa o perfil ao tipo correto: morador ou proprietário.
  • 3
    Permite vincular condomínio, unidade e endereço.
  • 4
    Guarda dados básicos para boletos, contratos e avisos.

Dica

Se o seu backend usa rotas diferentes, ajuste apenas as URLs no script: /api/auth/register para usuário e /api/proprietarios para proprietário.
